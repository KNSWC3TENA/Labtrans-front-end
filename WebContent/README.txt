Dificuldades Encontradas

-IDE utilizado para o desenvolvimento do projeto (Eclipse Jee Oxygen) possui certos bugs com java EE, mesmo utilizando as Libraries dentro
da pasta WEB-INF, certos erros custaram tempo de desenvolvimento do projeto os quais de outra maneira seriam indentificados mais cedo
ou não teriam a necessidade de reparo.

-Utilização inicial do JSF era de JSF 2.0, projeto precisou ser reiniciado como JSF 2.2.

-Para o desenvolvimento de uma API Restful, planos de implementação de métodos e beans baseados em sessões precisam ser descartados
completamente, o escopo do projeto não permite tal.

-Leves dúvidas sobre escolhas de nomes, aparências e posições de elementos no front-end, Banana é uma empresa fictícia a qual não se
há contato sobre preferências ou limitações em suas páginas.

-Desconhecimento inicial de sintaxe de Ajax atrapalhou um pouco o tempo de desenvolvimento do projeto.
